About Html-5

Html-5 is the latest version of the Hypertext Markup Language, the code that describes web pages.

Features of Html-5

  • New parsing rules for enhanced flexibility
  • Support for audio and video embedding
  • Improved support for web applications
  • New form controls and attributes

Getting Started with Html-5

To start using Html-5, you can use the following doctype declaration:

<!DOCTYPE html>

This declaration should be the first line in your HTML document.
